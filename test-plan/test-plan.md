# Test Plan - E-Commerce Testing

## Objective
To validate core functionalities of the e-commerce application including login, product selection, cart, and checkout process.

## Scope
In Scope:
- Login functionality
- Product listing
- Add to cart
- Checkout process

Out of Scope:
- Performance testing
- Security testing

## Test Strategy
Manual testing will be conducted based on predefined test cases. Automation testing will cover critical user flows.

## Test Environment
- Browser: Chrome
- OS: Windows 10

## Deliverables
- Test Plan
- Test Cases
- Bug Reports

## Risks
- Limited test data
- Third-party service dependency
